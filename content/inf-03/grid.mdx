CSS Grid to drugi (po Flexbox) system układu, ale działa w **dwóch wymiarach jednocześnie**: wiersze i kolumny. Gdy Flexbox wystarcza — używaj go. Gdy potrzebujesz pełnej siatki (np. layout strony, dashboard) — Grid jest królem.

## Flexbox vs Grid — kiedy czego

- **Flexbox** = 1D. Elementy w jednym rzędzie LUB jednej kolumnie. Idealny do: pasków nawigacji, list, przycisków, centrowania.
- **Grid** = 2D. Elementy na siatce wiersz × kolumna. Idealny do: layoutu strony (header/sidebar/main/footer), galerii, dashboardów, formularzy złożonych.

Nie ma jednej "lepszej" opcji — oba są dobre. Często łączysz: Grid na makro (layout strony), Flexbox wewnątrz (np. pasek nawigacji w headerze).

## Włączenie Grid

```css
.kontener {
    display: grid;
    /* lub: display: inline-grid; */
}
```

Domyślnie to jedna kolumna, tyle wierszy ile dzieci. Mało użyteczne — potrzebujemy zdefiniować siatkę.

## Definiowanie kolumn i wierszy

```css
.kontener {
    display: grid;
    grid-template-columns: 200px 1fr 200px;   /* 3 kolumny */
    grid-template-rows: auto 1fr auto;       /* 3 wiersze */
    gap: 16px;                                /* odstępy */
}
```

### Jednostki

- **`px`**, **`%`** — sztywne/procentowe.
- **`fr`** — "fraction", kawałek wolnej przestrzeni. `1fr 1fr 1fr` = 3 równe kolumny. `1fr 2fr` = pierwsza połowa, druga całość.
- **`auto`** — dopasuj do zawartości.
- **`minmax(min, max)`** — zakres. `minmax(200px, 1fr)` = minimum 200px, rośnie do wypełnienia.
- **`repeat(ile, wzorzec)`** — powtórzenie. `repeat(3, 1fr)` = trzy równe kolumny.
- **`auto-fit` / `auto-fill`** — tyle kolumn ile się zmieści.

```css
/* Responsywna galeria BEZ media queries */
.galeria {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 16px;
}
```

`auto-fit` z `minmax` daje **tyle kolumn ile się zmieści**, każda minimum 250px. Magia responsywności.

## Named areas — czytelne nazwy siatki

Zamiast numerów, nadajesz obszarom nazwy:

```css
.layout {
    display: grid;
    grid-template-columns: 200px 1fr;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
    gap: 16px;
    min-height: 100vh;
}

header  { grid-area: header; }
aside   { grid-area: sidebar; }
main    { grid-area: main; }
footer  { grid-area: footer; }
```

Wizualnie widać układ strony. Gdy trzeba dostosować — zmieniasz tylko `grid-template-areas`. Łatwe do refaktoru i zrozumienia.

## Umieszczanie dzieci w siatce

Trzy sposoby:

### 1. Automatyczny (domyślny)

Dzieci układają się w kolejności w wolne miejsca.

### 2. Numer linii

```css
.dziecko {
    grid-column: 1 / 3;    /* od linii 1 do linii 3 = 2 kolumny */
    grid-row: 2 / 4;       /* od linii 2 do linii 4 = 2 wiersze */
}
```

Linie są numerowane od 1. Skrót `grid-column: 1 / -1` oznacza "od początku do końca" (cała szerokość).

### 3. Named area (z kontenera)

```css
.dziecko {
    grid-area: header;     /* wpada w obszar "header" */
}
```

## Gap — odstępy

```css
.kontener {
    display: grid;
    gap: 16px;             /* wszędzie */
    column-gap: 24px;      /* tylko między kolumnami */
    row-gap: 12px;         /* tylko między wierszami */
    gap: 12px 24px;        /* wiersze kolumny */
}
```

`gap` w Grid działa tak samo jak we Flexbox — w przeciwieństwie do starych `margin` nie wymaga hacków na ostatnim elemencie.

## Wyrównanie dzieci wewnątrz komórki

```css
.kontener {
    display: grid;
    justify-items: stretch;     /* domyślne: rozciągnij do szerokości komórki */
    justify-items: center;      /* wyśrodkuj */
    justify-items: start;       /* do lewej */

    align-items: stretch;       /* domyślne: do wysokości */
    align-items: center;        /* wyśrodkuj */
    align-items: start;         /* do góry */
}

/* Dla konkretnego dziecka: */
.dziecko {
    justify-self: center;
    align-self: center;
}
```

## Justify/align content — wyrównanie CAŁEJ siatki

Gdy siatka jest mniejsza niż kontener:

```css
.kontener {
    display: grid;
    height: 100vh;
    grid-template-rows: repeat(3, 100px);

    justify-content: center;     /* wycentruj poziomo (jak w Flexbox) */
    align-content: center;       /* wycentruj pionowo */
}
```

## Kompletny przykład — layout strony

```html
<div class="layout">
    <header class="header">
        <img src="logo.png" alt="Logo">
        <nav>...</nav>
    </header>

    <aside class="sidebar">
        <h3>Moduły</h3>
        <ul>...</ul>
    </aside>

    <main class="main">
        <article>...</article>
    </main>

    <footer class="footer">
        © 2026 Koło STEM
    </footer>
</div>
```

```css
.layout {
    display: grid;
    grid-template-columns: 240px 1fr;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "header  header"
        "sidebar main"
        "footer  footer";
    gap: 16px;
    min-height: 100vh;
    padding: 16px;
}

.header  { grid-area: header;   display: flex; align-items: center; gap: 16px; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

@media (max-width: 768px) {
    .layout {
        grid-template-columns: 1fr;
        grid-template-areas:
            "header"
            "main"
            "sidebar"
            "footer";
    }
}
```

Na desktopie 2-kolumnowy (sidebar + main), na mobile jednokolumnowy (sidebar pod treścią).

<Callout type="exam">
Na egzaminie INF.03 zadania z CSS często wymagają Grid do layoutu strony. Zapamiętaj: `grid-template-areas` + `grid-area` na dzieciach + responsywność przez zmianę `grid-template-areas` w media queries.
</Callout>

## Kompletny przykład — dashboard z kartami

```html
<div class="dashboard">
    <div class="karta duza">1</div>
    <div class="karta">2</div>
    <div class="karta">3</div>
    <div class="karta">4</div>
    <div class="karta szeroka">5</div>
</div>
```

```css
.dashboard {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-auto-rows: 150px;
    gap: 16px;
}

.karta {
    background: #1a1a1a;
    padding: 16px;
}

.duza { grid-column: span 2; grid-row: span 2; }    /* 2x2 */
.szeroka { grid-column: span 3; }                    /* cała szerokość */
```

`grid-column: span 2` = zajmij 2 kolumny. `grid-row: span 2` = 2 wiersze.

## Podsumowanie

`display: grid` włącza Grid. `grid-template-columns` / `grid-template-rows` definiują siatkę. Jednostka `fr` dzieli wolną przestrzeń, `minmax()` daje zakres, `repeat(auto-fit, minmax(250px, 1fr))` robi responsywną galerię bez media queries. `grid-template-areas` + `grid-area` na dzieciach = czytelny layout. `gap` daje odstępy. Grid 2D (wiersze + kolumny), Flexbox 1D (rząd LUB kolumna) — łącz je świadomie.
