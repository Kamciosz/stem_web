## Zapytania SQL — gotowe rozwiązania

Baza: `studiotatuazu`. Plik SQL: `studiotatuazu.sql`.

### Zapytanie 1: Czarne tatuaże droższe niż 100 zł

```sql
SELECT wzor, cena FROM tatual WHERE kolor = (SELECT id FROM kolory WHERE nazwa = 'czarny') AND cena > 100 ORDER BY cena DESC;
```

### Zapytanie 2: Liczba tatuaży wg koloru (GROUP BY)

```sql
SELECT k.nazwa, COUNT(*) FROM tatual t JOIN kolory k ON t.kolor = k.id GROUP BY k.id;
```

### Zapytanie 3: Zielone tatuaże + styl (JOIN)

```sql
SELECT t.wzor, s.nazwa FROM tatual t JOIN style s ON t.styl = s.id WHERE t.kolor = (SELECT id FROM kolory WHERE nazwa = 'zielony');
```

### Zapytanie 4: Podwyżka o 10% dla czerwonych tatuaży

```sql
UPDATE tatual SET cena = cena * 1.1 WHERE kolor = (SELECT id FROM kolory WHERE nazwa = 'czerwony');
```
