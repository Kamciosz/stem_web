## Zapytania SQL — gotowe rozwiązania

### Zapytanie 1: czarne tatuaże powyżej 100 zł

```sql
SELECT wzor, cena FROM tatuaze
WHERE kolor = 'czarny' AND cena > 100
ORDER BY cena DESC;
```

### Zapytanie 2: liczba tatuaży wg koloru

```sql
SELECT kolor, COUNT(*) AS liczba FROM tatuaze
GROUP BY kolor;
```

### Zapytanie 3: zielone tatuaże z nazwą stylu (JOIN)

```sql
SELECT t.wzor, s.nazwa
FROM tatuaze t
JOIN style s ON t.id_styl = s.id
WHERE t.kolor = 'zielony';
```

### Zapytanie 4: podwyżka ceny o 10% dla czerwonych

```sql
UPDATE tatuaze SET cena = cena * 1.1
WHERE kolor = 'czerwony';
```

<Callout type="exam">
`UPDATE ... SET cena = cena * 1.1` — mnożenie przez 1.1 to podwyżka o 10%. Alternatywa: `cena + cena * 0.1`. Egzaminator sprawdza, czy cena naprawdę wzrosła (nie została nadpisana stałą wartością).
</Callout>

## Skrypt JavaScript — dodawanie wzoru do galerii

```javascript
function dodajWzor() {
    // Pobranie danych z kontrolek
    var wzor = document.getElementById("wzor").value;
    var plikInput = document.getElementById("plik");
    var kolor = document.getElementById("kolor").value;
    var cena = document.getElementById("cena").value;

    // Ustalenie nazwy pliku (bez ścieżki)
    var pelnaSciezka = plikInput.value;
    var nazwaPliku = pelnaSciezka.split("\\").pop();

    // Komunikat
    alert("Wzór: " + nazwaPliku + ", kolor " + kolor + " w cenie " + cena + " zł");

    // Utworzenie i dodanie obrazu do sekcji
    var img = document.createElement("img");
    img.src = nazwaPliku;
    img.alt = nazwaPliku;
    img.className = "miniatura";

    var sekcja = document.querySelector("section");
    sekcja.appendChild(img);
}
```

<Callout type="warning" title="Ścieżka pliku w input file">
`input[type=file].value` zwraca `C:\fakepath\plik.png`. Musisz wyciąć samą nazwę pliku: `split("\\").pop()` lub `split("/").pop()`. To pułapka z polecenia — jest nawet podpowiedź w arkuszu: "JS Value property: C:\fakepath\fileName.png".
</Callout>

## Wnioski z tego arkusza

1. **Arkusz mieszany PHP+JS** — baza w PHP, interakcja w JS. Ale strona to index.html (nie .php!) — skrypty PHP są tylko do zapytań w phpMyAdmin, nie do generowania strony.

2. **transition CSS** — jedyny arkusz z animacją CSS. Zapamiętaj składnię: `transition: property duration`.

3. **input file + accept** — `accept="image/png"` ogranicza wybór plików do PNG. Value zawiera `C:\fakepath\` — musisz to obciąć.

4. **createElement + appendChild** — tworzenie elementu DOM i dodawanie do sekcji. To najważniejsza umiejętność JS na tym egzaminie.

5. **UPDATE z obliczeniem** — `SET cena = cena * 1.1` to matematyka w SQL. Nie musisz pobierać wartości i wstawiać z powrotem — UPDATE robi to atomowo.

6. **@font-face** — tak jak w arkuszu 08 (Lato), ale tu Dancing Script. Dwa arkusze z własnymi czcionkami w jednej sesji.
