## O czym jest ten arkusz

Portal turystyczny Korony Gór Polskich — dwie strony PHP. index.php z odnośnikami do szczytów (skrypt 1) i miniaturami zdjęć (skrypt 2). szczyty.php z detalami szczytu (skrypt 3). Własna czcionka Lato z pliku. Efekt nth-child na span.

**Baza danych:** `korona` — tabele: `szczyty` (id, nazwa, wysokosc, pasmo, lokator, plik), `opis` (id, id_szczyt FK, tekst).

## Co musisz oddać — pliki

index.php + szczyty.php. Wspólne: header1 (logo.png) + header2 (h1 "Korona Gór Polskich"), sekcja (miniatury — skrypt 2), footer1 (Kontakt + email), footer2 (autor). Main index: nazwy szczytów jako odnośniki w span (skrypt 1). Main szczyty: obraz + nazwa + wysokość + pasmo + opis (skrypt 3).

## Grafika — co robić

logo.png: usunąć białe tło (przezroczystość), skadrować do prawego szczytu, przeskalować do 100px wysokości z zachowaniem proporcji.

## Zapytania SQL — gotowe rozwiązania

Baza: `korona`. Plik SQL: `korona.sql`.

### Zapytanie 1: szczyty posortowane malejąco wg wysokości

```sql
SELECT id, nazwa FROM szczyty ORDER BY wysokosc DESC;
```

### Zapytanie 2: pliki i nazwy pierwszych 10 szczytów

```sql
SELECT plik, nazwa FROM szczyty LIMIT 10;
```

### Zapytanie 3: szczyt id=1 z opisem (JOIN)

```sql
SELECT s.plik, s.nazwa, s.wysokosc, s.pasmo, o.tekst
FROM szczyty s
JOIN opis o ON s.id = o.id_szczyt
WHERE s.id = 1;
```

### Zapytanie 4: szczyty w Beskidach (LIKE)

```sql
SELECT nazwa, wysokosc, pasmo FROM szczyty
WHERE pasmo LIKE 'Beskid%';
```

<Callout type="tip" title="LIKE z %">
`LIKE 'Beskid%'` — znajduje wszystkie pasma zaczynające się od "Beskid" (Beskid Żywiecki, Beskid Śląski itd.). Znak `%` zastępuje dowolną liczbę znaków.
</Callout>

## Skrypty PHP — rozwiązanie krok po kroku

### Połączenie z bazą

```php
<?php
$conn = mysqli_connect("localhost", "root", "", "korona");
mysqli_query($conn, "SET CHARSET utf8");
```

### Skrypt 1 — nazwy szczytów jako odnośniki (index.php)

```php
$wynik = mysqli_query($conn,
    "SELECT id, nazwa FROM szczyty ORDER BY wysokosc DESC");
while ($row = mysqli_fetch_array($wynik)) {
    echo "<span><a href='szczyty.php?id=" . $row['id'] . "'>"
         . $row['nazwa'] . "</a></span>";
}
```

### Skrypt 2 — miniatury zdjęć

```php
$wynik2 = mysqli_query($conn, "SELECT plik, nazwa FROM szczyty LIMIT 10");
while ($row = mysqli_fetch_array($wynik2)) {
    echo "<img src='" . $row['plik'] . "' alt='" . $row['nazwa']
         . "' class='miniatura'>";
}
```

### Skrypt 3 — detale szczytu (szczyty.php)

```php
if (isset($_GET['id'])) {
    $id = $_GET['id'];
    $wynik3 = mysqli_query($conn,
        "SELECT s.plik, s.nazwa, s.wysokosc, s.pasmo, o.tekst
         FROM szczyty s JOIN opis o ON s.id = o.id_szczyt
         WHERE s.id = $id");
    $row = mysqli_fetch_array($wynik3);

    echo "<img src='" . $row['plik'] . "' alt='szczyt'>";
    echo "<h2>" . $row['nazwa'] . "</h2>";
    echo "<h3>wysokość: " . $row['wysokosc'] . " metrów n.p.m.</h3>";
    echo "<h3>pasmo górskie: " . $row['pasmo'] . "</h3>";
    echo "<p>" . $row['tekst'] . "</p>";
}
```

### Zamknięcie

```php
mysqli_close($conn);
?>
```

## Weryfikator w złym humorze

- czy pliki mają nazwy z polecenia, a nie twoje własne;
- czy formularz wysyła dane właściwą metodą i do właściwego pliku;
- czy PHP/JS uruchamia się dopiero wtedy, kiedy ma dane wejściowe;

- czy kod realizuje dokładnie polecenie, a nie 'podobny efekt'.

To jest miejsce, gdzie odpadają punkty za drobiazgi. Drobiazg w arkuszu to nadal wymaganie.

## Wnioski z tego arkusza

1. **Własna czcionka @font-face** — to jedyny arkusz w tej sesji z czcionką z pliku. Musisz umieć pisać `@font-face` z pamięci.

2. **LIKE z %** — wzorzec "Beskid%" to prosty pattern matching. Na egzaminie pojawia się regularnie.

3. **Dwie strony z przekazywaniem GET** — index.php → szczyty.php przez `?id=X`. Identyczny wzorzec jak w arkuszu 04 (matura).

4. **nth-child na span** — znowu efekt "chmury tagów" z różnymi rozmiarami. Tym razem więcej wartości (2,3,4,5,7,9,11,15,17,21,25,27).

5. **overflow: hidden na sekcji z miniaturami** — pasek miniatur ma stałą wysokość 90px. Nadmiar jest ukryty.
