## Zapytania SQL — gotowe rozwiązania

Baza: `korona`. Plik SQL: `korona.sql`.

### Zapytanie 1: Szczyty posortowane malejąco po wysokości

```sql
SELECT id, nazwa FROM szczyty ORDER BY wysokosc DESC;
```

### Zapytanie 2: Pliki i nazwy pierwszych 10 szczytów

```sql
SELECT plik, nazwa FROM szczyty LIMIT 10;
```

### Zapytanie 3: Dane + opis szczytu id=1

```sql
SELECT s.plik, s.nazwa, s.wysokosc, s.pasmo, o.opis FROM szczyty s JOIN opis o ON s.id = o.id_szczyt WHERE s.id = 1;
```

### Zapytanie 4: Szczyty w Beskidach (LIKE)

```sql
SELECT nazwa, wysokosc, pasmo FROM szczyty WHERE pasmo LIKE 'Beskid%';
```
