## Kwerendy z rozwiazania

```sql
Zapytanie 1: SELECT Data, Temat FROM szkolenia ORDER BY Data;
Zapytanie 2: SELECT Data, Temat, Nazwisko, Imie FROM szkolenia JOIN trenerzy ON trenerzy.Id = szkolenia.Id_trenera;
Zapytanie 3: SELECT Imie, Nazwisko, COUNT(*) FROM trenerzy JOIN szkolenia ON trenerzy.Id = Id_trenera GROUP BY trenerzy.Id;
Zapytanie 4: ALTER TABLE zapisy CHANGE COLUMN Id_klienta Id_sluchacza INT;
```
