## Zapytania SQL — gotowe rozwiązania

Baza: `kino`. Plik SQL: `kino.sql`.

### Zapytanie 1: Filmy trwające min. 100 min

```sql
SELECT id, tytul, rok, czas_trwania FROM filmy WHERE czas_trwania >= 100;
```

### Zapytanie 2: Dane aktora id=3

```sql
SELECT imie, nazwisko, plik FROM aktorzy WHERE id = 3;
```

### Zapytanie 3: Wszyscy aktorzy posortowani alfabetycznie

```sql
SELECT * FROM aktorzy ORDER BY nazwisko ASC, imie ASC;
```

### Zapytanie 4: Filmy z aktorem id=3 (JOIN n:m)

```sql
SELECT f.id, f.tytul, f.rok FROM filmy f JOIN filmy_aktorzy fa ON f.id = fa.id_film WHERE fa.id_aktor = 3;
```

### Zapytanie 5: Liczba filmów per aktor (GROUP BY)

```sql
SELECT id_aktor, COUNT(*) AS liczba_filmow FROM filmy_aktorzy GROUP BY id_aktor;
```
