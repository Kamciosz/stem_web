## Zapytania SQL — gotowe rozwiązania

Baza: `matura`. Plik SQL: `egzamin.sql`.

### Zapytanie 1: uczniowie ze szkoły T3

```sql
SELECT id, imie, nazwisko FROM maturzysta
WHERE szkola = 'T3'
ORDER BY nazwisko ASC;
```

### Zapytanie 2: unikalne przedmioty

```sql
SELECT DISTINCT przedmiot FROM arkusz;
```

### Zapytanie 3: najstarszy i najmłodszy arkusz

```sql
SELECT MIN(rok) AS 'Najstarszy', MAX(rok) AS 'Najmłodszy' FROM arkusz;
```

### Zapytanie 4: maturzysta z najlepszą średnią

```sql
SELECT id_maturzysta, AVG(punkty) AS Wynik
FROM wyniki
GROUP BY id_maturzysta
ORDER BY Wynik DESC
LIMIT 1;
```

<Callout type="exam">
Zapytanie 4 łączy GROUP BY + ORDER BY + LIMIT — trzy klauzule naraz. Alias `Wynik` w ORDER BY działa w MySQL/MariaDB (w innych bazach nie zawsze). Na egzaminie INF.03 używasz MySQL, więc jest OK.
</Callout>

### Zapytanie 5: wyniki maturzysty id=31 z JOIN

```sql
SELECT a.rok, a.sesja, a.przedmiot, w.punkty
FROM wyniki w
JOIN arkusz a ON w.id_arkusz = a.id
WHERE w.id_maturzysta = 31;
```

## Skrypty PHP — rozwiązanie krok po kroku

### Połączenie z bazą

```php
<?php
$conn = mysqli_connect("localhost", "root", "", "matura");
mysqli_query($conn, "SET CHARSET utf8");
```

### Skrypt 1 — 4 bloki statystyk (index.php i wynik.php)

```php
// Blok Przedmioty
$wynik = mysqli_query($conn, "SELECT DISTINCT przedmiot FROM arkusz");
echo "<div class='blok'><h4>Przedmioty</h4>";
while ($row = mysqli_fetch_array($wynik)) {
    echo $row['przedmiot'] . " ";
}
echo "</div>";

// Blok Lata
$wynik = mysqli_query($conn, "SELECT MIN(rok), MAX(rok) FROM arkusz");
$row = mysqli_fetch_array($wynik);
echo "<div class='blok'><h4>Lata</h4>" . $row[0] . " - " . $row[1] . "</div>";

// Blok Najlepszy wynik
$wynik = mysqli_query($conn,
    "SELECT AVG(punkty) AS Wynik FROM wyniki
     GROUP BY id_maturzysta ORDER BY Wynik DESC LIMIT 1");
$row = mysqli_fetch_array($wynik);
echo "<div class='blok'><h4>Najlepszy wynik</h4>" . round($row['Wynik']) . "%</div>";

// Blok Najgorszy wynik
$wynik = mysqli_query($conn,
    "SELECT AVG(punkty) AS Wynik FROM wyniki
     GROUP BY id_maturzysta ORDER BY Wynik ASC LIMIT 1");
$row = mysqli_fetch_array($wynik);
echo "<div class='blok'><h4>Najgorszy wynik</h4>" . round($row['Wynik']) . "%</div>";
```

### Skrypt 2 — lista uczniów jako odnośniki (index.php)

```php
echo "<h3>Wybierz ucznia z listy:</h3>";
$wynik = mysqli_query($conn,
    "SELECT id, imie, nazwisko FROM maturzysta WHERE szkola = 'T3' ORDER BY nazwisko ASC");
while ($row = mysqli_fetch_array($wynik)) {
    echo "<a href='wynik.php?id=" . $row['id'] . "'>"
         . $row['imie'] . " " . $row['nazwisko'] . "</a><br>";
}
```

### Skrypt 3 — wyniki wybranego maturzysty (wynik.php)

```php
if (isset($_GET['id'])) {
    $id = $_GET['id'];
    $wynik = mysqli_query($conn,
        "SELECT a.rok, a.sesja, a.przedmiot, w.punkty
         FROM wyniki w JOIN arkusz a ON w.id_arkusz = a.id
         WHERE w.id_maturzysta = $id");

    echo "<table><tr><th>Rok</th><th>Sesja</th><th>Przedmiot</th><th>Punkty</th></tr>";
    while ($row = mysqli_fetch_array($wynik)) {
        echo "<tr><td>" . $row['rok'] . "</td><td>" . $row['sesja'] . "</td>"
             . "<td>" . $row['przedmiot'] . "</td><td>" . $row['punkty'] . "</td></tr>";
    }
    echo "</table>";
}
```

### Zamknięcie

```php
mysqli_close($conn);
?>
```

## Wnioski z tego arkusza

1. **5 zapytań SQL zamiast 4** — ten arkusz ma o jedno zapytanie więcej. Zapytanie 4 (GROUP BY + AVG + ORDER + LIMIT) i zapytanie 5 (JOIN) to najtrudniejsze.

2. **Dwie strony PHP** — index.php i wynik.php. Przekazywanie id ucznia przez GET (`?id=31`) to klasyczny wzorzec.

3. **4 bloki z tą samą klasą** — egzaminator sprawdza, czy klasa CSS jest przypisana do wszystkich 4 bloków i czy `display: inline-block` (lub flex) ustawia je obok siebie.

4. **Modyfikacja zapytania 4** — skrypt 1 używa zapytania 4 dwa razy: raz z `DESC` (najlepszy), raz z `ASC` (najgorszy). To wymaga zmiany tylko jednego słowa w zapytaniu.
