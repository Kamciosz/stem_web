## O czym jest ten arkusz

Portal zgłoszeń wypadków — rejestr personelu (policjanci i ratownicy), pojazdów (karetki i radiowozy) i zgłoszeń. Jedna strona index.php z dwoma formularzami i dwoma sekcjami.

**Baza danych:** `zgloszenia` — tabele: `personel` (id, imie, nazwisko, status: policjant/ratownik), `pojazdy` (id, typ: K/R, marka), `rejestr` (id, id_personel FK, id_pojazd FK, data).

index.php: header (h1 "Zgłoszenia wydarzeń"), main z sekcją lewą (formularz radio Policjant/Ratownik + przycisk Pokaż + tabela skryptu 1) i sekcją prawą (h2 "Nowe zgłoszenie" + lista numerowana skryptu 2 + formularz z polem edycyjnym + przycisk "Dodaj zgłoszenie" skryptu 3), footer.

## Zapytania SQL — gotowe rozwiązania

Baza: `zgloszenia`. Plik SQL: `zgloszenia.sql`.

### Zapytanie 1: policjanci z personelu

```sql
SELECT id, imie, nazwisko FROM personel
WHERE status = 'policjant';
```

### Zapytanie 2: liczba pojazdów wg typu

```sql
SELECT typ, COUNT(*) AS liczba FROM pojazdy
GROUP BY typ;
```

### Zapytanie 3: osoby bez zgłoszeń (NOT IN)

```sql
SELECT id, nazwisko FROM personel
WHERE id NOT IN (SELECT id_personel FROM rejestr);
```

<Callout type="exam">
Podzapytanie `NOT IN` — to zaawansowany wzorzec. Alternatywnie można użyć `LEFT JOIN ... WHERE rejestr.id IS NULL`, ale NOT IN jest prostszy do napisania na egzaminie.
</Callout>

### Zapytanie 4: wstawienie rekordu z aktualną datą

```sql
INSERT INTO rejestr (id_personel, id_pojazd, data)
VALUES (1, 14, CURDATE());
```

<Callout type="tip" title="CURDATE()">
`CURDATE()` wstawia aktualną datę (format YYYY-MM-DD). Egzaminator sprawdza, czy użyłeś funkcji a nie wpisałeś datę na sztywno.
</Callout>

## Skrypty PHP — rozwiązanie krok po kroku

### Połączenie z bazą

```php
<?php
$conn = mysqli_connect("localhost", "root", "", "zgloszenia");
mysqli_query($conn, "SET CHARSET utf8");
```

### Skrypt 1 — tabela personelu wg opcji z formularza

```php
$opcja = isset($_POST['status']) ? $_POST['status'] : 'policjant';

echo "<h3>Wybrano opcję: $opcja</h3>";

$wynik = mysqli_query($conn,
    "SELECT id, imie, nazwisko FROM personel WHERE status = '$opcja'");

echo "<table><tr><th>Id</th><th>Imię</th><th>Nazwisko</th></tr>";
while ($row = mysqli_fetch_array($wynik)) {
    echo "<tr><td>" . $row['id'] . "</td><td>" . $row['imie'] . "</td>"
         . "<td>" . $row['nazwisko'] . "</td></tr>";
}
echo "</table>";
```

<Callout type="warning" title="Stan początkowy">
Polecenie mówi: "w stanie początkowym z opcją policjant". Dlatego `$opcja` ma domyślną wartość `'policjant'` gdy formularz nie został jeszcze wysłany.
</Callout>

### Skrypt 2 — lista osób bez zgłoszeń

```php
$wynik2 = mysqli_query($conn,
    "SELECT id, nazwisko FROM personel
     WHERE id NOT IN (SELECT id_personel FROM rejestr)");
while ($row = mysqli_fetch_array($wynik2)) {
    echo "<li>" . $row['id'] . " - " . $row['nazwisko'] . "</li>";
}
```

### Skrypt 3 — dodanie zgłoszenia

```php
if (isset($_POST['id_osoby'])) {
    $id_osoby = $_POST['id_osoby'];
    mysqli_query($conn,
        "INSERT INTO rejestr (id_personel, id_pojazd, data)
         VALUES ($id_osoby, 14, CURDATE())");
}
```

### Zamknięcie

```php
mysqli_close($conn);
?>
```

## Wnioski z tego arkusza

1. **Radio buttons (pola opcji)** — `input type="radio"` z `name` takim samym dla obu = tylko jedna opcja naraz. Domyślnie zaznaczony: atrybut `checked`.

2. **NOT IN z podzapytaniem** — zapytanie 3 jest trudne. Jeśli nie znasz NOT IN, przećwicz na danych testowych przed egzaminem.

3. **CURDATE()** — funkcja MySQL wstawiająca aktualną datę. Alternatywa: `NOW()` (data + czas), ale polecenie mówi o dacie.

4. **Dwa formularze na jednej stronie** — oba wysyłają POST do tego samego pliku. Skrypty muszą rozróżniać, który formularz wysłał dane (sprawdzanie `isset` odpowiedniego pola).

5. **Hover na przyciskach: zamiana kolorów** — tło Maroon → white, tekst white → Maroon. To odwrócenie kolorów.
