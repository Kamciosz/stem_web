## Zapytania SQL — gotowe rozwiązania

Baza: `zgloszenia`. Plik SQL: `zgloszenia.sql`.

### Zapytanie 1: Policjanci z personelu

```sql
SELECT id, imie, nazwisko FROM personel WHERE status = 'policjant';
```

### Zapytanie 2: Liczba karetek i radiowozów

```sql
SELECT typ, COUNT(*) FROM pojazdy GROUP BY typ;
```

### Zapytanie 3: Personel BEZ zgłoszeń (NOT IN)

```sql
SELECT id, nazwisko FROM personel WHERE id NOT IN (SELECT id_personel FROM rejestr);
```

### Zapytanie 4: Wstawienie zgłoszenia z aktualną datą

```sql
INSERT INTO rejestr (id_personel, id_pojazd, data) VALUES (1, 14, NOW());
```
