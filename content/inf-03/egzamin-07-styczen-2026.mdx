## Zapytania SQL — gotowe rozwiązania

Baza: `pogoda`. Plik SQL: `pogoda.sql`.

### Zapytanie 1: Średnia temperatura za lipiec zaokrąglona do 2 miejsc

```sql
SELECT ROUND(AVG(temperatura), 2) FROM pomiary WHERE id_miesiac = 7;
```

### Zapytanie 2: Wstawienie Kijowa

```sql
INSERT INTO miejscowosc (miasto, kraj) VALUES ('Kijów', 'Ukraina');
```

### Zapytanie 3: Miasta, kraje i temperatury w lipcu

```sql
SELECT m.miasto, m.kraj, p.temperatura FROM miejscowosc m JOIN pomiary p ON m.id = p.id_miejscowosc WHERE p.id_miesiac = 7;
```

### Zapytanie 4: Średnia temperatura dla każdego miesiąca

```sql
SELECT mi.nazwa, ROUND(AVG(p.temperatura),2) FROM miesiace mi JOIN pomiary p ON mi.id = p.id_miesiac GROUP BY mi.id;
```
