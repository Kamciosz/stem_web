## Zapytania SQL — gotowe rozwiązania

Baza: `choroby`. Plik SQL: `choroby.sql`.

### Zapytanie 1: Choroby zakaźne posortowane

```sql
SELECT nazwa FROM choroba WHERE zakazna = 'T' ORDER BY nazwa ASC;
```

### Zapytanie 2: Wszystkie choroby — id i nazwa

```sql
SELECT id, nazwa FROM choroba;
```

### Zapytanie 3: Objawy choroby id=5 przez relację

```sql
SELECT o.nazwa FROM objawy o JOIN choroba_objawy co ON o.id = co.id_objaw WHERE co.id_choroba = 5;
```

### Zapytanie 4: Nazwy chorób zakaźnych + liczba objawów

```sql
SELECT c.nazwa, COUNT(*) FROM choroba c JOIN choroba_objawy co ON c.id = co.id_choroba WHERE c.zakazna = 'T' GROUP BY c.id;
```
