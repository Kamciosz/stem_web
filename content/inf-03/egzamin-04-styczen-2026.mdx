## Zapytania SQL — gotowe rozwiązania

Baza: `matura`. Plik SQL: `egzamin.sql`.

### Zapytanie 1: Maturzyści ze szkoły T3 sortowani po nazwisku

```sql
SELECT id, imie, nazwisko FROM maturzysta WHERE szkola = 'T3' ORDER BY nazwisko ASC;
```

### Zapytanie 2: Unikalne nazwy przedmiotów

```sql
SELECT DISTINCT przedmiot FROM arkusz;
```

### Zapytanie 3: Najstarszy i najmłodszy arkusz

```sql
SELECT MIN(rok) AS najstarszy, MAX(rok) AS najmlodszy FROM arkusz;
```

### Zapytanie 4: Maturzysta z najlepszą średnią

```sql
SELECT id_maturzysta, AVG(punkty) AS Wynik FROM wyniki GROUP BY id_maturzysta ORDER BY Wynik DESC LIMIT 1;
```

### Zapytanie 5: Wyniki maturzysty 31 z JOIN

```sql
SELECT a.rok, a.sesja, a.przedmiot, w.punkty FROM arkusz a JOIN wyniki w ON a.id = w.id_arkusz WHERE w.id_maturzysta = 31;
```
