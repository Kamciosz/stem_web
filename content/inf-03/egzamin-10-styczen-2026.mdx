## Zapytania SQL — gotowe rozwiązania

Baza: `przepisy`. Plik SQL: `przepisy.sql`.

### Zapytanie 1: Dane potrawy id=7

```sql
SELECT nazwa, trudnosc, kalorie FROM potrawy WHERE id = 7;
```

### Zapytanie 2: Potrawa + rodzaj (JOIN 1:N)

```sql
SELECT p.nazwa, r.rodzaj FROM potrawy p JOIN rodzaje r ON p.id_rodzaj = r.id WHERE p.id = 7;
```

### Zapytanie 3: Potrawa + alergeny (relacja n:m przez 3 tabele)

```sql
SELECT p.nazwa, a.nazwa FROM potrawy p JOIN lista_alergenow la ON p.id = la.id_potrawa JOIN alergeny a ON la.id_alergen = a.id WHERE p.id = 7;
```

### Zapytanie 4: Przepis i plik graficzny potrawy id=7

```sql
SELECT przepis, plik FROM potrawy WHERE id = 7;
```
