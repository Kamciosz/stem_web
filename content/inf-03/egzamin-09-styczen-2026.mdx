## Zapytania SQL — gotowe rozwiązania

Baza: `paznokcie`. Plik SQL: `paznokcie.sql`.

### Zapytanie 1: Ważni klienci (topKlient=1)

```sql
SELECT imie, kolor, cena FROM klienci WHERE topKlient = 1;
```

### Zapytanie 2: Klienci z danego dnia

```sql
SELECT imie, cena FROM klienci WHERE data = '2024-05-23';
```

### Zapytanie 3: Klienci z kształtem migdał (JOIN)

```sql
SELECT k.imie, k.kolor, ks.nazwa FROM klienci k JOIN ksztalty ks ON k.ksztalt = ks.id WHERE ks.nazwa = 'migdał';
```

### Zapytanie 4: Dodanie kolumny doplata do tabeli wzory

```sql
ALTER TABLE wzory ADD COLUMN doplata INT;
```
