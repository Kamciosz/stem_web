### Schemat (dla przykładu: system rezerwacji sal)

```sql
CREATE DATABASE IF NOT EXISTS rezerwacje
    CHARACTER SET utf8mb4
    COLLATE utf8mb4_unicode_ci;

USE rezerwacje;

CREATE TABLE sale (
    id          INT AUTO_INCREMENT PRIMARY KEY,
    nazwa       VARCHAR(100) NOT NULL,
    pojemnosc   INT NOT NULL CHECK (pojemnosc > 0),
    opis        TEXT,
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE rezerwacje (
    id          INT AUTO_INCREMENT PRIMARY KEY,
    sala_id     INT NOT NULL,
    imie        VARCHAR(100) NOT NULL,
    email       VARCHAR(255) NOT NULL,
    data        DATE NOT NULL,
    godzina_od  TIME NOT NULL,
    godzina_do  TIME NOT NULL,
    status      ENUM('aktywna', 'anulowana') NOT NULL DEFAULT 'aktywna',
    created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (sala_id) REFERENCES sale(id) ON DELETE RESTRICT,
    CHECK (godzina_do > godzina_od),
    INDEX idx_rezerwacje_sala_data (sala_id, data)
);
```

## Faza 3: `db.php` (5 min)

```php
<?php
// db.php — jedyne miejsce z konfiguracją połączenia
$host = "localhost";
$dbname = "rezerwacje";
$user = "root";
$pass = "";

try {
    $pdo = new PDO(
        "mysql:host=$host;dbname=$dbname;charset=utf8mb4",
        $user,
        $pass,
        [
            PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
            PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
            PDO::ATTR_EMULATE_PREPARES => false
        ]
    );
} catch (PDOException $e) {
    // W produkcji: loguj, nie pokazuj szczegółów userowi
    die("Błąd połączenia z bazą danych: " . htmlspecialchars($e->getMessage()));
}
```

## Faza 4: `helpers.php` — walidacja i funkcje (10 min)

```php
<?php
// helpers.php
require_once __DIR__ . "/db.php";

function walidujRezerwacje(array $dane): array {
    $bledy = [];

    if (empty($dane["sala_id"])) {
        $bledy[] = "Wybierz salę.";
    } elseif (!is_numeric($dane["sala_id"])) {
        $bledy[] = "Nieprawidłowa sala.";
    }

    if (empty($dane["imie"])) {
        $bledy[] = "Imię jest wymagane.";
    } elseif (mb_strlen($dane["imie"]) < 2) {
        $bledy[] = "Imię musi mieć co najmniej 2 znaki.";
    }

    if (empty($dane["email"])) {
        $bledy[] = "E-mail jest wymagany.";
    } elseif (!filter_var($dane["email"], FILTER_VALIDATE_EMAIL)) {
        $bledy[] = "Nieprawidłowy format e-mail.";
    }

    $dzis = date("Y-m-d");
    if (empty($dane["data"])) {
        $bledy[] = "Data jest wymagana.";
    } elseif ($dane["data"] < $dzis) {
        $bledy[] = "Data musi być w przyszłości.";
    }

    if (empty($dane["godzina_od"]) || empty($dane["godzina_do"])) {
        $bledy[] = "Godziny są wymagane.";
    } elseif ($dane["godzina_do"] <= $dane["godzina_od"]) {
        $bledy[] = "Godzina zakończenia musi być po godzinie rozpoczęcia.";
    }

    return $bledy;
}

function czyWolna(PDO $pdo, int $salaId, string $data, string $od, string $do): bool {
    $stmt = $pdo->prepare("
        SELECT COUNT(*) FROM rezerwacje
        WHERE sala_id = :sala_id
          AND data = :data
          AND status = 'aktywna'
          AND (
              (godzina_od < :do AND godzina_do > :od)
          )
    ");
    $stmt->execute([
        ":sala_id" => $salaId,
        ":data" => $data,
        ":od" => $od,
        ":do" => $do
    ]);
    return $stmt->fetchColumn() == 0;
}

function pobierzSale(PDO $pdo): array {
    return $pdo->query("SELECT id, nazwa, pojemnosc, opis FROM sale ORDER BY nazwa")->fetchAll();
}

function pobierzRezerwacje(PDO $pdo, ?int $salaId = null): array {
    if ($salaId) {
        $stmt = $pdo->prepare("
            SELECT r.*, s.nazwa AS sala_nazwa
            FROM rezerwacje r
            JOIN sale s ON s.id = r.sala_id
            WHERE r.sala_id = :id
            ORDER BY r.data DESC, r.godzina_od DESC
        ");
        $stmt->execute([":id" => $salaId]);
    } else {
        $stmt = $pdo->query("
            SELECT r.*, s.nazwa AS sala_nazwa
            FROM rezerwacje r
            JOIN sale s ON s.id = r.sala_id
            ORDER BY r.data DESC, r.godzina_od DESC
        ");
    }
    return $stmt->fetchAll();
}
```

## Faza 7: Formularz rezerwacji (15-20 min)

```php
<?php
require_once "includes/db.php";
require_once "includes/helpers.php";

$title = "Nowa rezerwacja";

    "sala_id" => $_GET["sala_id"] ?? "",
    "imie" => "",
    "email" => "",
    "data" => "",
    "godzina_od" => "",
    "godzina_do" => ""
];

if ($_SERVER["REQUEST_METHOD"] === "POST") {
    $stare = array_merge($stare, $_POST);
    $bledy = walidujRezerwacje($stare);

    if (empty($bledy) && czyWolna($pdo, (int)$stare["sala_id"], $stare["data"], $stare["godzina_od"], $stare["godzina_do"]) === false) {
        $bledy[] = "Sala jest już zarezerwowana w tym terminie.";
    }

    if (empty($bledy)) {
        $stmt = $pdo->prepare("
            INSERT INTO rezerwacje (sala_id, imie, email, data, godzina_od, godzina_do)
```

## Podsumowanie

Workflow egzaminacyjny: DB → `db.php` → helpery → header/footer → strony → CSS → testy. **Zawsze** `db.php` na początku, **zawsze** prepared statements, **zawsze** `htmlspecialchars()` przy wyjściu. PRG (Post/Redirect/Get) po każdym INSERT/UPDATE/DELETE. Walidacja po obu stronach (HTML5 + PHP). Struktura plików: `includes/` + `css/` + strony. Testuj w przeglądarce: formularz, prepared statement, walidacja, mobilna responsywność. 30 minut na koniec = review kryteriów, nie nowe funkcje.
