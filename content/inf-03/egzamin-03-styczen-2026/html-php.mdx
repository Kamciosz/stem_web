Witryna matematyczna do obliczania pól figur. **Bez bazy danych** — ten arkusz sprawdza JavaScript, HTML i CSS.

Dwie strony: index.html (prostokąt i trójkąt — obliczenia JS) i kolo.html (animacja GIF + wzór).

Z klatki.jpg wycinasz 5 kwadratów 200x200px. Tworzysz animację GIF:
- 5 klatek, każda 200x200px
- Opóźnienie: 1 sekunda między klatkami
- Pętla: nieskończona

## JavaScript — skrypty (skrypt.js)

### Skrypt 1 — zmiana dużego obrazu po kliknięciu miniatury

```javascript
function zmienObraz(miniatura) {
    var duzy = document.getElementById("duzy");
    if (miniatura.src.includes("1m")) {
        duzy.src = "1d.bmp";
    } else {
        duzy.src = "2d.bmp";
    }
}
```

<Callout type="warning" title="Uwaga na src">
`miniatura.src` zwraca pełny URL (np. `file:///C:/...1m.bmp`), dlatego używamy `includes("1m")` zamiast porównania `==`. To częsty błąd na egzaminie — `== "1m.bmp"` nie zadziała.
</Callout>

### Skrypt 2 — obliczenie pola figury

```javascript
function oblicz() {
    var a = parseFloat(document.getElementById("bok1").value);
    var b = parseFloat(document.getElementById("bok2").value);
    var duzy = document.getElementById("duzy");
    var wynik = document.getElementById("wynik");

    if (duzy.src.includes("2d")) {
        // Prostokąt: P = a * b
        wynik.innerHTML = "Pole prostokąta = " + (a * b);
    } else {
        // Trójkąt: P = a * b / 2
        wynik.innerHTML = "Pole trójkąta = " + (a * b / 2);
    }
}
```

<Callout type="exam">
Polecenie mówi: "obliczenia powinny też działać dla stanu początkowego witryny (Skrypt 1 nie wykonał się ani razu), wtedy liczone jest pole trójkąta". Domyślnie duży obraz to 1d.bmp (trójkąt) — warunek `else` pokrywa ten przypadek. Egzaminator testuje to jako pierwszy krok.
</Callout>

## Wnioski z tego arkusza

1. **Arkusz bez bazy danych** — to rzadkość na INF.03 (zwykle jest PHP+SQL). Tutaj sprawdzane jest tylko JS/HTML/CSS + animacja GIF. Jeśli słabo znasz JS, ten arkusz może być trudniejszy niż te z PHP.

2. **Animacja GIF** — musisz umieć obsługiwać edytor grafiki rastrowej (GIMP lub Photoshop na egzaminie). Wycinanie klatek z jednego obrazu + ustawienie parametrów animacji to osobna umiejętność.

3. **includes() zamiast porównania** — `element.src` zwraca pełną ścieżkę URL, nie samą nazwę pliku. To pułapka, na którą wielu zdających wpada.

4. **Stan początkowy witryny** — egzaminator sprawdza, co się dzieje gdy od razu klikniesz "Oblicz" bez wybierania figury. Twój skrypt musi obsłużyć domyślny obraz (1d.bmp = trójkąt).

5. **Pseudo-element `::first-letter`** — pojawił się po raz pierwszy w sesji głównej. Zapamiętaj składnię i to, że działa tylko na elementach blokowych.
