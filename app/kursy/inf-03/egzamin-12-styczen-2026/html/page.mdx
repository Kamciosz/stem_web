## O czym jest ten arkusz

Studio tatuażu — arkusz mieszany PHP+JS. Strona index.html z galerią miniatur, formularzem dodawania wzoru (input file + select + input number), skryptem JS tworzącym nowe elementy DOM. Własna czcionka Dancing Script z pliku. Efekt hover z transition na miniaturach.

**Baza danych:** `studiotatuazu` — tabele: `tatauze` (id, wzor, kolor, cena, id_styl FK), `style` (id, nazwa).

## Co musisz oddać — pliki

index.html: header1 (logo.png), header2 (h1 "Studio Tatuażu"), main z section (h2 "Galeria" + 2 miniatury smok.png/koliber.png z klasą) i aside (h3 "Dodaj wzór" + formularz: label+input text, input file accept PNG, label+select kolory, label+input number cena, button "Dodaj Wzór" — onclick skrypt JS), footer. Grafika: logo.png — kadr ptaka + litery S/T czcionką ozdobną, 300px wysokości.
